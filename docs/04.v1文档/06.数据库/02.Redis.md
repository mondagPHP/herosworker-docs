# redis

对于 *redis* 操作使用 `framework\bootstrap\Redis` 类， redis 配置位于 `config/redis.php` 文件中， 直接 `Redis::方法` 使用， eg: `Redis::set('name', 'tomcat')`

> 如果不熟悉 *redis* 方法 参考： [操作](https://github.com/phpredis/phpredis)

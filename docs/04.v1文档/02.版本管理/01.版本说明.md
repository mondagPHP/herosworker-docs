# 版本说明

版本号采用 `x.y.z` 三位数，`x`代表主要版本， `y`代表次要版本， `z`代码小功能点迭代

- `x` 表示重大版本， 如果这个版本升级，一般需要参考**升级指南进**行对版本升级
- `y` 表示小版本的升级， 使用者对框架升级不需要考虑兼容性等问题
- `z` 表示小功能点的迭代或者bug的修复

> `z`版本不定时更新，`y`版本一个月发布一次， `x`版本暂未规划
